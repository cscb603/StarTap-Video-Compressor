# 🚀 星TAP 视频压缩 - 快速使用指南

## 👋 欢迎使用！

这是一个极简、高效的视频压缩工具，能帮你把视频变得更小！

---

## 🖱️ 图形界面版（推荐普通用户）

### 第一步：打开程序
双击 **「星TAP视频压缩v3.exe」** 即可打开！

### 第二步：添加视频
- **方式 1**：点击「➕ 添加文件」选择视频
- **方式 2**：点击「📂 添加文件夹」选择整个文件夹
- **方式 3**：直接把视频或文件夹拖到界面上！

### 第三步：选择档位（非常重要！）
**5 个预设档位，一键选择：**

| 档位 | 说明 | 推荐场景 |
|------|------|----------|
| 🎬 原始大小 - 不缩放 | 保持原分辨率，画质自定义 | 保留原始质量 |
| 📦 1080p (临时分享 · 极致压缩) | 1080p + CRF 32 + 高速 | 微信/临时分享，文件极小 |
| ⚖️ 1080p (推荐 · 平衡) | 1080p + CRF 24 + 中速 | **日常使用首选** |
| 📺 1440p (2K) - 高清 | 1440p + CRF 22 | 2K 高清质量 |
| 🎬 2160p (4K) - 原画 | 2160p + CRF 20 | 4K 原画画质 |

### 第四步：开始压缩！
点击「🚀 开始批量压缩」按钮！

## 💻 命令行版（给技术用户和 AI 工具）

### 快速压缩一个视频
```bash
fast-video-cli.exe --input 视频.mp4 --output-dir 输出文件夹
```

### 压缩整个文件夹
```bash
fast-video-cli.exe --input 视频文件夹 --output-dir 输出文件夹
```

### 临时分享模式（极速压缩）
```bash
fast-video-cli.exe --input 视频.mp4 --quick-share --max-height 1080 --quality 32 --speed-preset 10
```

### 选择编码器
```bash
# SVT-AV1（推荐，压缩率高，速度快）
fast-video-cli.exe --input 视频.mp4 --encoder svt-av1

# NVENC（NVIDIA 显卡，极快！）
fast-video-cli.exe --input 视频.mp4 --encoder hevc-nvenc

# x265（兼容性最好）
fast-video-cli.exe --input 视频.mp4 --encoder x265
```

### 调整最大尺寸
```bash
# 原始大小（不缩放）
fast-video-cli.exe --input 视频.mp4 --max-height 0

# 1080p
fast-video-cli.exe --input 视频.mp4 --max-height 1080

# 2K (1440p)
fast-video-cli.exe --input 视频.mp4 --max-height 1440

# 4K (2160p)
fast-video-cli.exe --input 视频.mp4 --max-height 2160
```

### 查看所有参数
```bash
fast-video-cli.exe --help
```

## 📱 关于 AV1 格式播放

- **新设备**（Win11 22H2+/macOS 13+/Android 14+/iOS 17+）：原生播放
- **老设备**：用 **VLC 播放器**（下载地址：https://www.videolan.org/vlc/）
- **B站/YouTube**：完美支持，推荐上传 AV1 格式

## ❓ 常见问题

**Q: 压缩后的视频在哪里？**
A: 如果你没选导出目录，就在原视频旁边，文件名会加 `_s` 后缀。

**Q: 为什么有些视频无法播放？**
A: AV1 格式需要新设备或 VLC 播放器。

**Q: 压缩速度太慢怎么办？**
A: 如果你有 NVIDIA 显卡，选 NVENC 编码器！极快！

**Q: 可以同时压缩多个视频吗？**
A: 可以！你可以添加多个视频，程序会自动处理！

**Q: 竖屏视频会怎么处理？**
A: 程序会自动判断！竖屏视频会限制宽度而不是高度，保持画面比例！
